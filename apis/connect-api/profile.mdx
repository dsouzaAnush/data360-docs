Connect REST API endpoints for Data 360 profile:

## Operations

| Method | Path | Summary | Available Version |
| --- | --- | --- | --- |
| `GET` | `/ssot/profile/metadata` | Get profile metadata | 52.0 |
| `GET` | `/ssot/profile/metadata/{dataModelName}` | Get profile data model object metadata | 52.0 |
| `GET` | `/ssot/profile/{dataModelName}` | Get profile data model object | 52.0 |
| `GET` | `/ssot/profile/{dataModelName}/{id}` | Get profile data model object with search key | 52.0 |
| `GET` | `/ssot/profile/{dataModelName}/{id}/calculated-insights/{ciName}` | Get profile data model object and calculated insight with search key | 52.0 |
| `GET` | `/ssot/profile/{dataModelName}/{id}/{childDataModelName}` | Get profile data model object and child object with search key | 52.0 |
