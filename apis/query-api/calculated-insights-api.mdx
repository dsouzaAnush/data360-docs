# Query Calculated Insights

The Calculated Insights API provides a powerful way to query and analyze data from Data 360 Calculated Insights objects. This API allows you to retrieve metadata about available insights and query the insights with selected SQL dimensions and measures, including optional filters and time-based aggregations.

-   **Metadata Access**: Retrieve information about available calculated insights, including dimensions, measures, and relationships
-   **Flexible Querying**: Query insights with customizable dimensions, measures, and filters
-   **Pagination Support**: Handle large datasets with limit and offset parameters
-   **Time-based Analysis**: Aggregate data by different time granularities (HOUR, DAY, MONTH, QUARTER, YEAR)
-   **Sorting Capabilities**: Order results using multiple fields in ascending or descending order

-   Maximum return limit: 4,999 rows per query
-   Pagination support via limit and offset parameters
-   Comprehensive error handling with detailed response codes

-   [**GET /api/v1/insight/metadata**](https://developer.salesforce.com/docs/data/data-cloud-query-guide/references/data-cloud-query-api-reference/c360a-api-v1-insight-metadata.html)

    -   Retrieves metadata for all available calculated insights
    -   Returns dimensions, measures, and relationships for each insight
-   [**GET /api/v1/insight/metadata/\{ci-name\}**](https://developer.salesforce.com/docs/data/data-cloud-query-guide/references/data-cloud-query-api-reference/c360a-api-insights-meta-ci-name.html)

    -   Retrieves metadata for a specific calculated insight
    -   Returns detailed information about dimensions, measures, and relationships

-   [**GET /api/v1/insight/calculated-insights/\{ci-name\}**](https://developer.salesforce.com/docs/data/data-cloud-query-guide/references/data-cloud-query-api-reference/c360a-api-insights-ci-ci-name.html)
    -   Query a specific calculated insight with customizable parameters
    -   Supports filtering, sorting, pagination, and time-based aggregation
    -   Returns actual data values based on specified dimensions and measures
