## API Quick Reference

| API | Base Path | Auth |
|-----|-----------|------|
| **Query API** | `POST /services/data/v64.0/ssot/query-sql` | OAuth 2.0 + DC token exchange |
| **Profile API** | `GET /api/v1/profile/{entityName}` | OAuth 2.0 + DC token exchange |
| **Ingestion API (Streaming)** | `POST /api/v1/ingest/sources/{sourceName}/{objectName}` | OAuth 2.0 + DC token exchange |
| **Ingestion API (Bulk)** | `POST /api/v1/ingest/jobs` | OAuth 2.0 + DC token exchange |
| **Connect REST API** | `/services/data/v64.0/ssot/*` | Standard Salesforce OAuth |
| **Metadata API** | Standard Metadata API | Standard Salesforce OAuth |

## Object Naming Conventions

| Type | Suffix | Example |
|------|--------|---------|
| Data Model Object | `__dlm` | `UnifiedIndividual__dlm` |
| Standard DMO field | `ssot__*__c` | `ssot__FirstName__c` |
| Calculated Insight Object | `__cio` | `CustomerLTV__cio` |
| Custom field | `__c` | `LifetimeValue__c` |
