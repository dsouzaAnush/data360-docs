# Developer Guide

This section covers the tools and APIs available for programmatic access to Data 360. For conceptual platform guides (ingestion, segmentation, identity resolution), see [Platform Guides](/developer-guide/data-ingestion-guide).

## APIs

| API | Purpose | Start Here |
|-----|---------|------------|
| **Connect REST API** | Manage data streams, segments, activations, identity resolution | [Overview](/apis/connect-api/index) |
| **Query API** | Execute SQL queries, retrieve profiles, metadata, calculated insights | [Overview](/apis/query-api/index) |
| **Ingestion API** | Push data via REST (streaming JSON or bulk CSV) | [Ingestion Guide](/integrations/ingestion-api-guide) |
| **Metadata API** | Deploy Data 360 configuration across orgs | [Metadata Types](/apis/metadata-types/index) |

## Apex Integration

Query Data 360 DMOs directly from Apex using SOQL (v61.0+) or `ConnectApi.CdpQuery` for full SQL:

```java icon=java
// Static SOQL with DMOs
List<UnifiedIndividual__dlm> profiles = [
    SELECT ssot__Id__c, ssot__FirstName__c, ssot__Email__c
    FROM UnifiedIndividual__dlm
    WHERE LifetimeValue__c > 5000
    LIMIT 100
];
```

See [Apex Integration](/developer-guide/apex-integration) for the full guide including CdpQuery, ConnectApi, and testing patterns.
