# Data 360 Connect REST API

Use Salesforce Data 360 Connect REST API to build custom solutions on top of Data 360 and take advantage of uniform implementation and testing practices.

[Get Started](https://developer.salesforce.com/docs/data/connectapi/guide)

[API Reference](https://developer.salesforce.com/docs/data/connectapi/references)

## Key Features

-   **Query and retrieve data** - Run SQL queries, search profiles, look up metadata, and retrieve customer data from data lake objects (DLOs), data model objects (DMOs), segments, and calculated insights.
-   **Configure data pipelines and models** - Create and manage DMOs, data streams, connections, identity resolution rulesets, and data transforms to build unified customer profiles.
-   **Activate audiences and automate actions** - Build segments, activate audiences across channels, trigger data actions, and deploy predictive models to drive customer engagement.
