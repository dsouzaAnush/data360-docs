# Event Specifications

Event specifications for the Engagement Mobile SDK.

[**Cart Event**](https://developer.salesforce.com/docs/data/data-cloud-engagement-mobile-sdk/guide/c360a-api-engagement-mobile-sdk-cart-event.html)

Use a cart event interaction to capture when a customer modifies the contents of their online shopping cart.

[**Catalog Event**](https://developer.salesforce.com/docs/data/data-cloud-engagement-mobile-sdk/guide/c360a-api-engagement-mobile-sdk-catalog-event.html)

Use a catalog event interaction to capture when a customer interacts with various tracking items. For example, a product or a blog post.

[**Order Event**](https://developer.salesforce.com/docs/data/data-cloud-engagement-mobile-sdk/guide/c360a-api-engagement-mobile-sdk-order-event.html)

An order event interaction is an ecommerce event that captures actions performed by a customer to manage new, in progress, or completed orders.

[**Custom Event**](https://developer.salesforce.com/docs/data/data-cloud-engagement-mobile-sdk/guide/c360a-api-engagement-mobile-sdk-custom-event.html)

To capture user-defined attributes in the mobile SDK, define the source properties in a custom event. You can then map the fields to a Custom Event Schema using the Mobile Schema naming conventions.

[**Line Item Attribute**](https://developer.salesforce.com/docs/data/data-cloud-engagement-mobile-sdk/guide/c360a-api-engagement-mobile-sdk-line-item-attribute.html)

Line items are intended to describe purchasable items and are used in cart and order interactions.

[**Order Attribute**](https://developer.salesforce.com/docs/data/data-cloud-engagement-mobile-sdk/guide/c360a-api-engagement-mobile-sdk-order-attribute.html)

An order attribute is contained within the event structure of Order Event and mapped in accordance with the order schema.
