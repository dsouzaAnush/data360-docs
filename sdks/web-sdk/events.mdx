# Event Specifications

The Salesforce Interactions SDK supports several event types for capturing customer engagement on your website. Each event has a defined structure that maps to Data 360 DMOs through the Web Connector schema.

## Event Structure

Every event contains system-set properties and optional custom properties:

```javascript icon=js
SalesforceInteractions.sendEvent({
  interaction: {
    name: 'Event Name',          // Required: descriptive name
    eventType: 'eventType',      // Optional: categorization
    // ... event-specific properties
  },
  user: { /* optional user data */ },
  account: { /* optional account data */ }
});
```

| Property | Type | Auto-Set | Description |
|----------|------|----------|-------------|
| `interaction.name` | string | No | Descriptive event name |
| `interaction.eventType` | string | No | Event categorization |
| `timestamp` | datetime | Yes | Event time (auto-generated) |
| `anonymousId` | string | Yes | First-party cookie ID |
| `pageUrl` | string | Yes | Current page URL |
| `referrerUrl` | string | Yes | Referring page URL |

## Cart Interaction

Captures when a customer modifies their shopping cart — add, remove, or update items.

```javascript icon=js
SalesforceInteractions.sendEvent({
  interaction: {
    name: 'Add to Cart',
    lineItem: {
      catalogObjectType: 'Product',
      catalogObjectId: 'SKU-12345',
      quantity: 2,
      price: 129.99,
      currency: 'USD'
    }
  }
});
```

### Line Item Properties

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| `catalogObjectType` | string | Yes | Object type (e.g., "Product") |
| `catalogObjectId` | string | Yes | Unique product identifier |
| `quantity` | number | Yes | Number of items |
| `price` | number | No | Unit price |
| `currency` | string | No | ISO 4217 currency code |

**Maps to:** Cart DMO, Cart Product DMO

## Catalog Interaction

Captures when a customer views or interacts with catalog items — product pages, blog posts, search results.

```javascript icon=js
SalesforceInteractions.sendEvent({
  interaction: {
    name: 'Product Viewed',
    catalogObject: {
      type: 'Product',
      id: 'SKU-12345',
      attributes: {
        name: 'Running Shoes',
        price: 129.99,
        category: ['Footwear', 'Athletic'],
        brand: 'Northern Trail',
        imageUrl: 'https://example.com/shoes.jpg'
      }
    }
  }
});
```

**Maps to:** Website Engagement DMO

## Order Interaction

Captures ecommerce order events — purchase completion, order status changes.

```javascript icon=js
SalesforceInteractions.sendEvent({
  interaction: {
    name: 'Order Completed',
    order: {
      id: 'ORD-67890',
      totalValue: 259.98,
      currency: 'USD',
      lineItems: [
        {
          catalogObjectType: 'Product',
          catalogObjectId: 'SKU-12345',
          quantity: 2,
          price: 129.99
        }
      ]
    }
  }
});
```

**Maps to:** Sales Order DMO, Sales Order Product DMO

## User Data

Send profile data for a known or anonymous user, with or without an associated interaction:

```javascript icon=js
SalesforceInteractions.sendEvent({
  user: {
    identities: {
      emailAddress: 'jane@example.com',
      customerId: 'CUST-001'
    },
    attributes: {
      firstName: 'Jane',
      lastName: 'Doe',
      loyaltyTier: 'Gold'
    }
  }
});
```

**Maps to:** Individual DMO, Contact Point Email DMO

## Account Data

Capture account-level data for B2B scenarios:

```javascript icon=js
SalesforceInteractions.sendEvent({
  account: {
    id: 'ACCT-001',
    attributes: {
      name: 'Acme Corp',
      industry: 'Technology',
      employeeCount: 500
    }
  }
});
```

**Maps to:** Account DMO

## Consent Data

See [Consent API](/sdks/web-sdk/consent) for the full consent event specification.

## Custom Events

For events that don't fit standard types, use custom events:

```javascript icon=js
SalesforceInteractions.sendEvent({
  interaction: {
    name: 'Survey Completed',
    eventType: 'custom',
    attributes: {
      surveyId: 'SURV-001',
      score: 9,
      category: 'NPS'
    }
  }
});
```

Custom events require a matching custom schema in your Web Connector configuration.
